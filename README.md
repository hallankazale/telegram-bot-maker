# Telegram Bot Maker (Bot-Mestre)

Este bot é um Bot-Mestre que cria e registra novos bots no Telegram.

## Como rodar

1. Configure a variável de ambiente:

MASTER_BOT_TOKEN=SEU_TOKEN_AQUI

2. Para rodar:

pip install -r requirements.txt python bot.py

## Comandos

- `/start` – inicia o bot
- `/addbot TOKEN_DO_BOT` – registra um novo bot (placeholder para funcionalidade futura)

Salve.
